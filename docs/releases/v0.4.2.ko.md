# agentcli v0.4.2

[English](v0.4.2.md) | [한국어](v0.4.2.ko.md)

Codex 첫 turn bootstrap 동작을 보정한 패치 릴리즈입니다.

## 설치

GitHub-first 릴리즈:

```bash
pip install "agentcli @ git+https://github.com/saintiron82/agentcli.git@v0.4.2"
```

향후 PyPI 릴리즈:

```bash
pip install agentcli
```

## 수정

- Codex JSONL 파서가 네이티브 CLI bootstrap 인사문
  `Ready. What would you like me to work on?`을 작업 결과로 취급하지 않습니다.
- 새 Codex 세션의 첫 응답이 그 인사문뿐이면, 생성된 Codex thread를 resume해서
  같은 prompt를 한 번만 다시 전달합니다.

이제 첫 `client.chat(..., provider="codex")` 호출에서 bootstrap 인사문이 작업 결과로
반환되는 문제를 피합니다.
