# agentcli v0.4.0

[English](v0.4.0.md) | [한국어](v0.4.0.ko.md)

`agentcli`는 사용자의 컴퓨터에 설치된 Claude Code, Codex, GitHub Copilot CLI를 Python 앱의 AI 백엔드로 임베드하기 위한 beta client layer로 정리되었습니다.

## 설치

GitHub-first 릴리즈:

```bash
pip install "agentcli @ git+https://github.com/saintiron82/agentcli.git@v0.4.0"
```

향후 PyPI 릴리즈:

```bash
pip install agentcli
```

## 주요 내용

- Claude Code, Codex, GitHub Copilot CLI를 하나의 `LLMClient` API로 호출합니다.
- 네이티브 CLI 세션이 히스토리의 source of truth입니다. `agentcli`는 session handle, usage row, routing metadata만 저장합니다.
- `owner + alias + cwd`로 장기 실행 app-level agent를 식별합니다.
- provider/model을 명시적으로 선택하고 fallback은 opt-in으로만 동작합니다.
- health check는 UI/log에 안전한 출력 형태를 제공합니다.
- streaming error를 provider와 무관하게 표준화합니다.
- `AgentProfile.materialize()`는 사용자 작성 파일을 보호하면서 관리형 `AGENTS.md`와 `CLAUDE.md`를 씁니다.

## 상태

이 릴리즈는 public beta입니다. 개발자 통합과 피드백에는 적합하지만 1.0 수준의 안정성 약속은 아닙니다. 각 사용자는 Claude Code, Codex, GitHub Copilot CLI를 별도로 설치하고 인증해야 합니다.
