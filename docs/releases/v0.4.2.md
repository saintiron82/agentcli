# agentcli v0.4.2

[English](v0.4.2.md) | [한국어](v0.4.2.ko.md)

Patch release for Codex first-turn bootstrap behavior.

## Install

GitHub-first release:

```bash
pip install "agentcli @ git+https://github.com/saintiron82/agentcli.git@v0.4.2"
```

Future PyPI release:

```bash
pip install agentcli
```

## Fixes

- Codex JSONL parsing now ignores the native CLI bootstrap greeting:
  `Ready. What would you like me to work on?`
- If a new Codex session returns only that greeting, agentcli retries the same
  prompt once by resuming the newly created Codex thread.

This keeps the first `client.chat(..., provider="codex")` call from treating the
bootstrap greeting as the task result.
